JavaScript

Изначально JavaScript был создан, чтобы «сделать веб-страницы живыми».Программы на этом языке называются скриптами. Они могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы.Скрипты распространяются и выполняются, как простой текст. Им не нужна специальная подготовка или компиляция для запуска.Это отличает JavaScript от другого языка – Java.

Язык №1

JavaScript являетcя самым популярным языков программирования в мире. Его место в рейтинге не просто так, ведь его использование занимаент 19.1% среди всех остальных. Он не так сложен в начинании и может подойти для начинающих веб-программистов. 

О преимуществах и недостатках языка

Преимущеста+
Интерпретируемый язык. Код работает сразу же в браузере, не требуется каких-либо затрат для его компиляции. Это, несомненно, плюс: не нужно ничего ждать, результат кода можно сразу же увидеть в браузере. Востребован​ный язык программирования. Простота и рациональность применения.

Недостатки-
Недостатки JavaScript: Отсутствие чтения и загрузки файлов. Это сделано из соображения безопасности для пользователей. Доступен для злоумышленников. Весьма легко встроить какой-либо вредоносный код, который может нанести большой урон. Динамическая типизация. Причина частых проблем при разработке – нет возможности выявить ошибки заранее, только на этапе работы.
